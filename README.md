# 🌿 College Zero Waste Management
**23GN01C · Innovation Through Design Thinking**

A production-ready website for the College Zero Waste Management initiative at AR Nagar Engineering College, featuring a **live real-time waste tracker**.

## 🚀 Deploy to Vercel (2 minutes)

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → **New Project**
3. Import your GitHub repo
4. Leave all settings as default → click **Deploy**
5. Done! Your site is live ✅

## 📁 Files

| File | Purpose |
|------|---------|
| `index.html` | Entire website (single file, no dependencies) |
| `vercel.json` | Vercel deployment configuration |

## ✨ Features

- **Live Waste Tracker** — real-time kg counter for Recyclable, Non-Recyclable, and Compostable waste, estimated from 6 AM each day based on ~500 kg/day campus data
- **Live Clock** — hours, minutes, seconds elapsed since campus operations began
- **Department Bar Chart** — actual measured recyclable waste data by department (EEE, MECH, CSE, etc.)
- **Impact Assessment** — Environmental, Economic, Societal, Cultural impacts
- **Solutions Section** — 6 design-thinking-based proposals
- **Team Section** — all 4 team members

## 📊 Waste Rate Basis

| Type | % of Daily 500 kg | kg/min approx |
|------|-------------------|---------------|
| Recyclable | 25% | 0.149 kg/min |
| Non-Recyclable | 45% | 0.268 kg/min |
| Compostable | 30% | 0.179 kg/min |

Data sourced from field measurements and department surveys conducted as part of the Design Thinking course.
